# DSpace Angular Migration Route

:::info Objective
Transfer technical and strategic capabilities for upgrading **DSpace 5.x/6.x** repositories to versions **7, 8, or 9**.
:::

## ðŸ“… Course Structure

### Module 1: Migration Planning and Strategy
:::tip Ideal for
Library directors, repository managers, project leaders, and technical coordinators.
:::

**Contents:**
- **Customization Inventory:** Identification of native functions vs. bespoke developments.
- **Infrastructure Requirements:** Hardware and software needed (Docker vs. Traditional).
- **Version Analysis:** Key differences between DSpace 7.x, 8.x, and 9.x.
- **Change Management:** Roadmap for socialize the change with researchers and end users.

> **CONTENT:** Module 1 Introduction_Course DSpace 2026

---

### Module 2: The New Backend (Server and API)
:::note Ideal for
System administrators and developers.
:::

**Contents:**
- **DSpace Angular Architecture:** Separation between Server (REST API) and Client (Interface).
- **Core Installation and Configuration:** Step-by-step backend guide.
- **Solr Configuration:** Migration and optimization of indices and statistics.
- **Security and Authentication:** Protocols (LDAP, Shibboleth, ORCID).

---

### Module 3: The Angular Frontend (Interface and UX)
:::note Ideal for
Frontend developers and web designers.
:::

**Contents:**
- **Angular Basics:** Components, routes, and themes.
- **Visual Customization (Theming):** Application of institutional branding.
- **Interface Configuration:** Homepage, menus, and item visualization.
- **Multilingualism:** Managing labels in Spanish and Portuguese.

---

### Module 4: Data and Metadata Migration
:::caution Ideal for
Catalogers and data technicians.
:::

**Contents:**
- **Database Upgrade:** DB schema migration scripts and processes.
- **Bitstream Migration:** Integrity verification and physical transfer.
- **Metadata Cleaning:** Preparatory normalization.
- **Collection Mapping:** Hierarchy restructuring.

---

### Module 5: Interoperability and Advanced Features
:::info Ideal for
Visibility and metrics specialists, repository managers.
:::

**Contents:**
- **OpenAIRE 4 Configuration:** Alignment with LA Referencia.
- **Entities and Relationships:** People, Organizations, and Projects (DSpace 7+).
- **Statistics and Analytics:** Matomo and native visualizers.
- **Persistent Identifiers:** Handles, DOIs, and ARK preparation.

---

### Module 6: Autonomy and Community Support
:::tip Ideal for
The entire repository team.
:::

**Contents:**
- **Using the GitHub Board:** Bug reporting and help requests.
- **AI-Assisted Consultations:** Project chatbots and NotebookLM.
- **Live Documentation:** Navigation through ES/PT manuals.
- **Upstream Contribution:** Sharing improvements with the global community.
